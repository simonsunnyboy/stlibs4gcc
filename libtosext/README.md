# TOS extension library

(c) 2025 by Matthias Arndt <marndt@final-memory.org>

This library has dependencies to TOS functions and its goal is to extend 
Atari TOS functionality where useful within programs.

It is not meant as a TOS replacement.

It extends GEMDOS file handling.

## How to use

Compile and link the source files from src/ as necessary.
  
## Testing

Test executables for Atari ST can be build from test/
